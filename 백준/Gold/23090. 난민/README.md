### 문제 설명

<p>A나라에서 내전이 일어났다!</p>

<p>이 때문에 많은 내전 난민들이 B나라로 오게 되었고, B나라는 이들을 위해 난민촌을 건설해 주었다.</p>

<p>다음은 난민촌을 좌표평면으로 나타낸 그림이다. 파란색 선은 강, 빨간색 점은 난민의 거주 위치를 나타낸다.</p>
